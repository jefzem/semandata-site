Une plateforme sémantique non-statistique qui encode vos règles métier en langage naturel et les expose aux agents IA via API REST. Résultat déterministe. Zéro hallucination possible. Powered by Data Excellence Science® — 19 ans de R&D.

Principe fondateur

Déterministe, souverain, agentique.

DEMS® n'est pas une application. C'est une plateforme. La différence : toutes ses fonctionnalités sont exposées via API REST et consommables par n'importe quel agent IA, n'importe quel système, en temps réel.

  • ✦Non-statistique — le moteur SMM® ne fait pas d'inférence. Il raisonne sur des règles explicites. Résultat binaire : conforme ou non conforme. Pas de probabilité. Pas d'hallucination.
  • ✦Souverain by design — DEMS® est déployé dans votre environnement. Vos données et règles ne quittent jamais votre périmètre. Ni vers GDE, ni vers Semandata, ni vers aucun tiers.
  • ✦Agentique natif — conçu pour l'ère des agents avant que le terme existe. BER®, KVI®, DEI® : tout est interrogeable en REST. Vos agents décident dans le cadre que vous avez défini.
  • ✦SaaS géré — mises à jour transparentes. Aucune charge d'administration. SLA contractuel sur 3 niveaux.

Les modules DEMS®

Six modules. Une IP déposée. Un référentiel universel.

🧠

SMM® — Semantic Meta Model

L'ontologie universelle des données d'entreprise. Le SMM® crée une langue commune entre tous vos systèmes hétérogènes — SAP, Salesforce, ERP custom, bases de données legacy. Chaque entité, relation et attribut reçoit un sens partagé, défini par le métier, pas par l'informatique.

  • ✦ Moteur non-statistique — résultat déterministe ou absence de résultat. Zéro hallucination structurellement impossible.
  • ✦ Alignement automatique des entités sémantiques entre systèmes disparates.
  • ✦ Compatible SAP nativement (30 000 tables connues), Oracle, Microsoft Dynamics, Salesforce.
  • ✦ Fondement de toute la couche BER®, KVI® et DEI®.

BER® — Business Excellence Requirements

Les règles métier de l'entreprise en langage naturel. Interprétables sans traduction technique par les dirigeants, les juristes, les contrôleurs — ET par les agents IA. Une règle modifiée par le métier est propagée instantanément à tous les agents via API.

  • ✦ Édition en langage naturel — aucune compétence technique requise pour créer ou modifier une règle.
  • ✦ Versioning complet — chaque modification est tracée, horodatée, réversible.
  • ✦ Propagation instantanée — une règle modifiée est active sur tous les agents en temps réel.
💶

KVI® — Key Value Indicators

Ce qui n'est pas conforme a un prix. Les KVI® expriment la non-conformité en euros — pas en pourcentage de complétude. C'est la différence entre un rapport data et une décision de Comex. Le risque invisible devient un actif gérable.

  • ✦ Valorisation monétaire de chaque écart de conformité — en temps réel, chaque matin.
  • ✦ Hiérarchisation automatique des priorités par valeur à risque.
  • ✦ Dashboards Comex — la gouvernance devient une décision business, pas IT.
  • ✦ ROI moyen constaté sur POV : 15 à 30× l'investissement.
